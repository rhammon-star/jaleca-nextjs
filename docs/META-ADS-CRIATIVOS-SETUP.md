## 1️⃣ REMARKETING DINÂMICO — Carrossel Catálogo

### 📍 Onde Criar

**Meta Ads Manager** → Campanhas → **"Remarketing Dinâmico — Jaleca Abr 2026"** → Ad Set **"Visitantes 30 dias"** → **+ Criar Anúncio**

---

### 🎨 Formato

**Carrossel Dinâmico de Catálogo**
- Mostra produtos que o usuário visualizou
- Até 10 cards por carrossel
- Facebook/Instagram puxa do catálogo automaticamente

---

### ✍️ Texto do Anúncio

**Texto Principal (125 caracteres):**
```
Você viu isso aqui — e ainda está disponível 👀

Volte e finalize sua compra. Jalecos premium com frete grátis no Sudeste.
```

**Título (40 caracteres):**
```
Veja os Produtos que Você Visualizou
```

**Descrição (opcional, 30 caracteres):**
```
3x sem juros · Frete Grátis
```

---

### 🔗 URL de Destino

**Padrão (automático via catálogo):**
- Facebook/Instagram direcionam para a página do produto visualizado

**Parâmetros UTM (já configurados no feed):**
```
utm_source=facebook&utm_medium=remarketing&utm_campaign=dinamico
```

---

### 🎯 Botão de Ação (CTA)

Escolher: **"Comprar agora"**

---

### ⚙️ Configurações

1. **Formato:** Carrossel
2. **Fonte do catálogo:** Jaleca (ID do catálogo Meta)
3. **Conjunto de produtos:** Todos os produtos
4. **Ordenação:** Mais visualizados primeiro
5. **Número de cards:** Até 10
6. **Design:** Template automático (deixar Facebook otimizar)

---

### 📋 Checklist Remarketing Dinâmico

## 2️⃣ NOVAS PÁGINAS — Imagem Única

### 📍 Onde Criar

**Meta Ads Manager** → Campanhas → **"Novas Páginas — Jaleca Abr 2026"** → Ad Set **"Público Frio"** → **+ Criar Anúncio**

---

### 🎨 Formato

**Imagem Única** (Feed + Stories)
- Foco: Jaleco preto feminino (produto com melhor conversão)
- Público: Frio (nunca visitou o site)

---

### 🖼️ Imagem Necessária

**Especificações Técnicas:**
- **Formato Feed:** 1080 x 1080 px (quadrado 1:1)
- **Formato Stories:** 1080 x 1920 px (vertical 9:16)
- **Formato:** JPG ou PNG
- **Tamanho máximo:** 30 MB
- **Texto na imagem:** Máximo 20% da área

**Conteúdo da Imagem:**
- Jaleco preto feminino (Slim ou modelo premium)
- Modelo usando (não manequim)
- Fundo clean (branco/bege/cinza claro)
- Foco no caimento e qualidade do tecido

**Onde encontrar:**
- Usar foto profissional existente do jaleco preto
- Exportar em 1080x1080 (Feed) + 1080x1920 (Stories)
- Produto sugerido: Jaleco Slim Preto Feminino

---

### ✍️ Texto do Anúncio

**Texto Principal (125 caracteres):**
```
Antes de você se apresentar, seu jaleco já foi avaliado.

Vista o que você merece. Tecido premium, caimento impecável.
```

**Título (40 caracteres):**
```
Jalecos Jaleca — Compre Agora
```

**Descrição (30 caracteres):**
```
3x sem juros · Frete Grátis
```

---

### 🔗 URL de Destino

**URL final:**
```
https://jaleca.com.br/jaleco-feminino?utm_source=facebook&utm_medium=prospecting&utm_campaign=novas_paginas
```

Ou direcionar para produto específico:
```
https://jaleca.com.br/produto/jaleco-slim-preto-feminino?utm_source=facebook&utm_medium=prospecting&utm_campaign=novas_paginas
```

---

### 🎯 Botão de Ação (CTA)

Escolher: **"Comprar agora"**

---

### ⚙️ Configurações

**Posicionamentos:**
- ✅ Feed Facebook
- ✅ Feed Instagram
- ✅ Stories Instagram

- ✅ Reels (se disponível)

**Otimização:**

## 📊 Métricas para Monitorar

### Remarketing Dinâmico (R$15/dia)
- **CTR desejado:** >1.5% (remarketing geralmente >1%)
- **CPA desejado:** <R$50
- **ROAS desejado:** >3.0x
- **Conversões esperadas:** 1-2/semana

### Novas Páginas (R$6/dia)
- **CTR desejado:** >0.8% (público frio geralmente 0.5-1%)
- **CPA desejado:** <R$70
- **ROAS desejado:** >2.5x
- **Conversões esperadas:** 0-1/semana (público frio demora mais)

---

## ⚠️ Importante

1. **Texto em imagem:** Meta penaliza anúncios com >20% de texto na imagem
2. **Testes A/B:** Após 3 dias com dados, duplicar anúncio e testar texto alternativo
3. **Público Frio vs Remarketing:** Remarketing sempre tem métricas melhores
4. **Paciência:** Público frio precisa 7-14 dias para otimizar
