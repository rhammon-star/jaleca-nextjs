# Google Display - Criativo Remarketing

## 📍 Onde Criar

**Google Ads** → Campanhas → **"Remarketing Display"** (R$15/dia) → Ad Group **"Visitantes — Jaleca"** → Anúncios → **+ Novo anúncio**

---

## 🎨 Tipo de Anúncio

**Display Responsivo**
- Google combina automaticamente suas imagens + textos
- Testa diferentes variações para encontrar a melhor performance

---

## 🖼️ Imagens Necessárias

### Especificações Técnicas
- **Paisagem:** 1200 x 628 px (ratio 1.91:1)
- **Quadrado:** 1200 x 1200 px (ratio 1:1)
- **Formato:** JPG ou PNG
- **Tamanho máximo:** 5 MB por imagem
- **Mínimo:** 1 imagem de cada tipo

### Imagens Sugeridas (5 no total)

**Paisagem (1200x628):**
1. Hero do site - jaleco slim branco fundo neutro
2. Jaleco preto feminino - modelo usando
3. Coleção de jalecos coloridos - grid com 4-6 produtos

**Quadrado (1200x1200):**
4. Jaleco slim tradicional feminino - close-up caimento
5. Badge "Frete Grátis SP/RJ/MG/ES" + produto

**Onde encontrar:**
- Usar fotos já existentes do site em `/public/`
- Exportar do Figma/Canva em alta resolução
- Pedir ao designer para criar versões otimizadas

## 📋 Checklist de Criação

- [ ] 1. Acessar Google Ads → Remarketing Display → Visitantes — Jaleca
- [ ] 2. Clicar em "+ Novo anúncio" → Display responsivo
- [ ] 3. Fazer upload de 3 imagens paisagem + 2 quadrado
- [ ] 4. Adicionar 5 títulos curtos
- [ ] 5. Adicionar 2 títulos longos (opcional)
- [ ] 6. Adicionar 4 descrições
- [ ] 7. Definir URL de destino com UTM
- [ ] 8. Escolher CTA "Comprar agora"
- [ ] 9. Pré-visualizar em diferentes formatos
- [ ] 10. Salvar e publicar

---

## 🎨 Dicas de Criação

1. **Consistência de marca:** Use as cores Jaleca (bege/nude, preto, branco)
2. **Foco no produto:** Jaleco deve ocupar 60%+ da imagem
3. **Evite texto em imagem:** Google já adiciona seus títulos/descrições
4. **Teste A/B:** Google roda todas combinações e mostra as melhores
5. **Qualidade:** Fotos profissionais convertem 3x mais que stock photos

---

## 📊 Monitoramento

Após publicar, monitorar:
- Impressões (meta: >10k/semana)
- CTR (meta: >0.5%)
- Conversões (meta: >3/semana com R$15/dia)
- CPA (meta: <R$35)

Ajustar criativos após 7 dias de dados.
