# Novas ideias estratégicas adicionadas ao backlog

## #42 — MCP Server SEO programático
Servidor MCP que gera páginas para todas as palavras-chave do projeto Jaleca de forma automatizada. Hub topical em escala — uma página otimizada por keyword, com schema, internal linking e copy gerada via IA.

**Por quê:** escalar topical authority sem trabalho manual página a página.

## #43 — Ferramenta viral grátis "Me veja com o jaleco"
Try-on / preview virtual com IA. Usuário sobe foto e ferramenta gera imagem dele(a) vestindo um jaleco do catálogo. Captura lead + viraliza por compartilhamento orgânico nas redes.

**Por quê:** aquisição barata via UGC, diferencial competitivo, alimenta funil.

## #44 — Auditoria SEO + AEO + Schema em todas as páginas
Garantir que toda página tenha:
- SEO técnico (title, meta, H1, internal links, canonical)
- AEO (FAQ, conteúdo direto para Answer Engines / AI Overviews)
- Schema completo conforme tipo: Product, FAQPage, HowTo, Article, BreadcrumbList, AggregateRating

**Por quê:** maximizar share-of-voice em SERP + LLMs/AI Overviews.
